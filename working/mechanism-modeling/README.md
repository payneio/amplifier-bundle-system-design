# Modeling the behavior of a bundle

Use the [Behavioral spec recipe](../../recipes/behavioral-spec.md) to model the behavior of a bundle. The recipe takes a bundle as input and produces a formal specification of its behavior, capturing the interactions between its mechanisms and the expected outcomes.
